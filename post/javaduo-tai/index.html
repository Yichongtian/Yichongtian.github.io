多态

多态的前提是：两个对象（类）存在继承关系

多态的向上转型

  1. 本质：父类的引用指向了子类的对象
  2. 语法：父类类型 引用名 = new 子类类型();
  3. 特点：编译类型看左边，运行类型看右边
    可以调用父类中的所有成员（须遵守访问权限），
    不能调用子类中特有成员
    运行的结果由子类的实现决定

多态的向下转型

  1. 语法：子类类型 引用名 = (子类类型)父类引用;
  2. 特点：只能强转父类的引用，不能强转父类的对象
    要求父类的引用必须指向的是当前目标类型的对象
    //先向上转型，此时animal的编译类型为Animal，运行类型为Cat，无法调用cat特有的方法和属性
//调用子类重写父类的方法时，子类的实现生效
Animal animal = new Cat();

//如果想要这个animal也可以调用cat的方法，就需要用到向下转型
//此时cat的编译类型和运行类型都是Cat
//向下转型只能发生在这种情况，animal这个父类引用指向的其实是Cat类型的对象，所以才能强转
Cat cat = (Cat) animal;
    向下转型后，可以调用子类类型中的所有成员
